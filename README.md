# pure-css-sidebar-project
A responsive sidebar navigation menu built using HTML &amp; CSS with a pure CSS toggle (no JavaScript).
## Zing Studio Sidebar Menu
## Description
This is a simple and responsive sidebar navigation menu built using HTML and CSS only.
The project demonstrates how to create an interactive sidebar using a checkbox toggle technique, without using JavaScript

## Features
Responsive sidebar design
Pure CSS toggle (no JavaScript used)
Smooth transitions and hover effects
Navigation menu with icons
Social media icons included

## Technologies Used
HTML5
CSS3
Google Fonts
Font Awesome Icons

## How It Works
The sidebar is controlled using a hidden checkbox input.
When the checkbox is checked → sidebar appears
When unchecked → sidebar hides
This technique allows interaction without JavaScript.

## How to Run
1.Download or clone this repository
2.Open the project folder
3.Run the index.html file in your browser
